# Simple Interest Calculator

This project is a simple interest calculator implemented using a Bash script.

## Purpose

The purpose of this project is to calculate simple interest based on user input.

## How It Works

The script takes three inputs:

* Principal amount (P)
* Rate of interest (R)
* Time period (T)

It calculates simple interest using the formula:
SI = (P × R × T) / 100

## How to Run

1. Make the script executable:
   chmod +x simple-interest.sh

2. Run the script:
   ./simple-interest.sh

## Example

Input:
P = 1000
R = 5
T = 2

Output:
Simple Interest = 100
